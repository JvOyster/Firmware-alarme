# Firmware-alarme
```
1º passo: instale o git
2º passo: abra o vscode
3º passo: na parte de cima onde ficam as opções file, edit, run etc..., encontre a opção terminal, clique e depois clique em new terminal
4º passo: quanbdo o terminal abrir, localize um sinal de "+", passe o mouse por cima e aparecerá uma setinha pra baixo, clique nessa setinha, e depois clique em git bash
5º passo:vc verificará que no canto direito do terminal, terá 2 opções, a de cima sendo powershell, e a de baixo sendo o git bash que você abriu antriormente, passe o mouse por cima da opção powershell e a delete apertando num icone de lixeira.
6º passo: digite no terminal o comando: "git init"
7º passo: agora digite o comando "git status" e você irá perceber que não tem arquivos adicionados
8º passo: digite o comando "git add ."
9º passo: digite o comando "git status" novamente e vc perceberá que um arquivo foi adicionado
10º passo: digite o comando "git commit -m "qualquer nome que quiser"
11º passo> digite o comando "git branch -M main"
12º passo: digite o comando "git remote add origin https://github.com/JvOyster/Firmware-alarme"
13º passo: digite o comando "git push -u origin main "
```
